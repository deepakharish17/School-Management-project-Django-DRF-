# School Management System - Django & DRF

A backend focused School Management System built using Django, Django REST Framework, MySQL, and Pandas.

This project started as a way to learn Django Models and CRUD operations, but gradually evolved into a complete backend application capable of managing schools, students, and academic records through REST APIs.

During development, I explored core Django concepts such as Models, Migrations, ORM Queries, Foreign Keys, Custom Decorators, API Logging, Excel Processing, and Database Relationships. The project helped me understand how real-world backend systems handle data validation, bulk uploads, error handling, and API monitoring.
